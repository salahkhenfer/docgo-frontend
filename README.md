# healthpathglobal Frontend

## Environment

Copy `.env.example` to `.env`.

- `VITE_API_URL`: backend base URL (scheme + domain + optional path, no trailing slash)
  - `https://backend.healthpathglobal.com`
  - `https://backend.healthpathglobal.com`
  - `https://healthpathglobal.com/api`

## Run locally

- `npm install`
- `npm run dev`
